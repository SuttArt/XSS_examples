Example 1: Reflected XSS
How to:
If a user search for: < script>alert('XSS')< /script>, the script will be executed immediately, causing an XSS vulnerability.



Example 2: Persistent XSS
How to:
If a malicious user submits a comment like: < script>alert('XSS Stored!')< /script>, any user visiting Comments Page will see the alert, indicating stored XSS.


    Example 3: DOM-Based XSS
    How to:
    If userInput contains: < img src=x onerror="alert('DOM XSS')">, the script will be executed, demonstrating a DOM-based XSS vulnerability.
